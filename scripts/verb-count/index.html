Verb Counts

:: 2 min read (412 words)

This script uses the Get-Verb command to sum and provide the percentage of each verb in the Powershell environment. It prints a sorted table with both the individual verb’s number of commands and what percentage it makes up.


$counts = @{}
$verbs = (Get-Verb).Verb
$Start = (Get-Date)

foreach ($verb in $verbs) { $counts[$verb] = (gcm -Verb $verb).Count }
$Sum = ($counts.Values | Measure-Object -Sum).Sum

$counts.GetEnumerator() | Sort-Object -Property Value
                        | Select-Object Key,Value,@{Name = 'Percentage'; Expression = {(($_.Value) / $Sum).ToString("P")}}
                        | Format-Table -AutoSize

$End = (Get-Date)
Write-Host "$($Sum) commands counted in $(($End - $Start).TotalMilliseconds) ms"
